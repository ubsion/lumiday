# Lumiday Website

Static GitHub Pages website for Lumiday（闪光日子）, a beautifully designed countdown app for life’s important moments.

## Pages

- `index.html` - product landing page with Chinese / English language toggle
- `support.html` - support contact and FAQ
- `privacy.html` - App Store friendly privacy policy
- `styles.css` - shared Luminous Minimal visual system
- `script.js` - language switching and subtle reveal interactions
- `favicon.svg` - lightweight Lumiday browser icon

## Preview

Open `index.html` directly in a browser, or serve the repository root with any static server.

## GitHub Pages

Deploy from the repository root on the `main` branch.

Expected URL:

`https://<github-username>.github.io/lumiday/`
